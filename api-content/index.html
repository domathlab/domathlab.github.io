终于有一本书，不全是数学题，而是从人的成长的角度讲述，为什么要学数学。《数学的力量》这本书的英文名是Mathematics For Human Flourish，直译过来就是“数学使人类繁荣”。什么可以使人类繁荣？爱、公正、永恒、自由、美、探索…… 第二本要推荐的是《怎样解题》。学数学当然要做数学题，一道好的数学题可以带来很纯粹的思维享受。在作者乔治·波利亚的眼中，解题是一门艺术，他追求清晰而简洁的描述。 “为了真正地了解一个理论，你必须知道它是如何被发明的。”数学的思考过程应该是启发式的。解题就是解谜、揭密，一旦一个问题被有逻辑地解决，于解题者而言，这个问题就具有了永恒性，这是多么大的喜悦。 作者提炼出解题的基本步骤：1、理解题目；2、建立联系；3、拟定并执行方案；4、回顾、检查、讨论。解一道题，和做一个工程项目，有相似的流程。不只是做题，我们做很多事，岂不都是这个思考过程？这也是为什么，我们要学数学。 我要推荐的第三本书是《数学简史》。我对知识的起源充满好奇心，在研究数学教育时，我希望了解到具体的数学知识是从何而来的、在历史中又是如何发展的。于是，我开始学习数学的历史。关于数学的历史，我受两位老师影响很大，一位是华东师范大学汪晓勤老师的《HPM：数学史与数学教育》，王晓勤老师的课程让我知道，数学史原来是一个学科专业，很多的数学知识都还在不断地被溯源。另一位则是浙江大学蔡天新教授的《数学简史》，蔡教授既是数论方面的专家，又是诗人，他的《数学简史》读起来，既可以清晰地看见数学对人类文明的伟大贡献，又可以感受到文字的诗意优美，就像随着蔡教授在世界各地旅行。 如果你想了解数学与人类文明的关系，这本《数学简史》应该是你推开的第一扇门。 我推荐的最后一本书是《塑造数学脑》。我更喜欢这本书的副标题：用数学思维解决非数学问题。我问过很多人，都说数学在生活中无处不在，那么你可以告诉我，数学到底在哪里吗？回答最多的是两种答案：1、数字在生活中无处不在，到处都有数量；2、图形在生活中无处不在，到处都是形状。 这些回答当然没有错，但数学对生活的影响远不止于此。**这本书真正做到了，从生活中看见数学。**庭院里的小径、景区的地图、足球比赛等，都可以通过由浅入深地思考，从生活的特殊案例到数学的抽象化理解，欣赏到数学的真正魅力。数学的欣赏是有门槛的，你要真的愿意去思索、探索、求索，建立真正的理解，数学就会变成一首美妙的歌曲、一幅充满想象的画作、一座令人称奇的建筑。 02 推荐人 吴莹
